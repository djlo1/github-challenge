Portail du Test Pratique — Programme de Stage

Règles & objectifs

Bienvenue. Ce test a deux phases :

  1. Accès collaboratif aux documents chiffrés (phase commune)
  2. Travail individuel sur vos épreuves (après décryptage)

Important :

Les fichiers sont chiffrés avec GPG (AES-256). Vous devez d’abord trouver et récupérer les fichiers chiffrés, puis les déchiffrer localement pour commencer vos tâches.

Étapes à suivre

  1. Lisez entièrement cette page.
  2. Trouvez l’emplacement du premier fichier chiffré (indice : inspectez le code source de la page).
  3. Téléchargez le fichier chiffré correspondant (ai.pdf.gpg).
  4. Pour le second fichier, un indice est caché dans une image fournie sur ce site. Analysez ses métadonnées.
  5. Une fois les mots de passe trouvés, déchiffrez localement : 
    gpg --output ai.pdf --decrypt ai.pdf.gpg
gpg --output si.pdf --decrypt si.pdf.gpg
  6. Ouvrez chaque PDF et commencez vos tâches.

Règles de fair-play

Vous pouvez coopérer pour la phase d’accès mais chaque candidat travaille seul pour sa partie individuelle. Toute triche ou tentative de contournement non autorisée annule la participation.

Indices & Aides

Indice principal : si vous ne voyez pas immédiatement les liens, pensez à Voir le code source (Ctrl+U) et cherchez les commentaires HTML. Un lien relatif vers le fichier chiffré y est glissé.

Indice secondaire (image) : téléchargez l'image visible sur ce site et vérifiez ses métadonnées (EXIF).
